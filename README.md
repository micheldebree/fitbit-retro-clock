# fitbit-retro-clock

Retro clockface for fitbit Versa

## Development

### Prerequisites

- NodeJS
- Yarn
- Fitbit Emulator

### Install dependencies

```sh
yarn install
```

### Generate new appid

If you are making a new version of the clock face, make sure to generate its own
appid:

```sh
yarn run generate-appid
```

### Build

```sh
yarn run build
```

### Test

```sh
yarn run test
```
